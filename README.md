# marlonapps

# Primeros pasos en Django
```
$ python3 - venv env
$ pip install django
$ pip freeze
$ django-admin startproject marlonapps .
$ python3 manage.py runserver
$ pip freeze > requirements.txt
```

# Activar el entorno
```
$ source ./env/bin/activate
```

# Correr server
```
$ python3 manage.py runserver
```

# Recolectar staticos
```
$ python manage.py collectstatic
```
